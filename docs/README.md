# OctoAcme Project Management Docs

This README serves as the entry point for OctoAcme's project management knowledge base. It provides a concise summary of our processes and links to all major process documents to help teammates — new and existing — quickly orient themselves and find what they need.

## About OctoAcme's Project Management Approach

OctoAcme follows a customer-first, iterative delivery model across all cross-functional projects. Every project has a named Project Manager (PM) and Product Lead to ensure clear ownership, accountability, and coordination. Decisions are data-informed, and the team prioritizes psychological safety to encourage open feedback and continuous learning.

Work is organized in repeatable increments with defined milestones and backlogs. Risk management is built into the process through risk registers and structured escalation paths, keeping stakeholders informed at all stages. After each delivery, retrospectives capture learnings and feed action items back into process improvements.
